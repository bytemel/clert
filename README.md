🏗 Project Structure
src/
├── components/       # Reusable components
│   ├── layout/       # Page wrappers
│   ├── sections/     # Page sections
│   └── ui/           # Custom UI primitives
├── lib/              # Utilities & hooks

├── assets/           # Images & fonts

🚀 Getting Started
Install dependencies

bash
npm install
Run development server

bash
npm run dev
Build for production

bash
npm run build
